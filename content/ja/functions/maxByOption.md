# `maxByOption`

{{< signature maxByOption >}}

`maxByOption` は、各要素に `f` を適用していき、最大の値を生成する元の要素を `Some` に包んで返します。

@include [figure.html source="../images/maxByOption.svg" desc="関数 maxByOption の図"]

空のコレクションでは、この関数は `None` を返します。
